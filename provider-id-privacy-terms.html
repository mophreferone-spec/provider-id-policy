ประกาศความเป็นส่วนตัว (Privacy Notice)

Provider ID
ระบบเว็บไซต์พอร์ทัล เป็นระบบบริการแพลตฟอร์มข้อมูลเจ้าหน้าที่ผู้ให้บริการทางการแพทย์ โดยมีการใช้ Provider ID ซึ่งถือเป็นการยืนยันตัวตนในรูปแบบอิเล็กทรอนิกส์ของบุคลากรทางการแพทย์ ผ่านระบบเว็บไซต์ พอร์ทัล เพื่อให้บุคลากรทางการแพทย์สามารถเข้าถึงข้อมูลผู้ป่วยได้อย่างสะดวกและรวดเร็ว ตามนโยบาย “บัตรประชาชนใบเดียว รักษาได้ทุกที่”

กระทรวงสาธารณสุข จะดำเนินการตามพระราชบัญญัติคุ้มครองข้อมูลส่วนบุคคล พ.ศ. 2562 และกฎหมายอื่นที่เกี่ยวข้อง ซึ่งได้มีข้อกำหนดในการให้บริการ จึงขอประกาศความเป็นส่วนตัว (Privacy Notice) ให้กับผู้ใช้บริการ ทั้งนี้ ขอให้ผู้ใช้บริการศึกษารายละเอียดและเงื่อนไข ของประกาศฉบับนี้โดยละเอียด

คำนิยาม

ผู้ให้บริการ Provider ID ได้รับมอบหมาย โดยกระทรวงสาธารณสุขในการบำรุงรักษา พัฒนา และกำหนดรายละเอียดของ Provider ID ตามพระราชบัญญัติการบริหารงานและการให้บริการภาครัฐผ่านระบบดิจิทัล พ.ศ. 2562

ผู้ใช้บริการ หมายถึง บุคคลธรรมดา นิติบุคคล หรือหน่วยงานใด ๆ ที่มีการเข้าใช้บริการ Provider ID

บริการ Provider ID หมายถึง การใช้ผลิตภัณฑ์ ซอฟต์แวร์ และเว็บไซต์ของผู้ให้บริการ เพื่อจัดเก็บ รวบรวม เผยแพร่ข้อมูล ตามพระราชบัญญัติการบริหารงานและการให้บริการภาครัฐผ่านระบบดิจิทัล พ.ศ. 2562

เจ้าหน้าที่ผู้ให้บริการ หมายถึง บุคลากรทางการแพทย์ที่มีหน้าที่ให้บริการผู้ใช้บริการ Provider ID

ข้อกำหนดในการให้บริการของผู้ให้บริการ (ด้านความเป็นส่วนตัว)

  • ผู้ให้บริการจะใช้ข้อมูลส่วนบุคคลที่ได้รับจากผู้ใช้บริการเพื่อการให้บริการและพัฒนา Provider ID เท่านั้น
  • ข้อมูลส่วนบุคคลจะถูกเก็บรักษาอย่างปลอดภัยและเป็นไปตามกฎหมายที่เกี่ยวข้อง
  • ไม่มีข้อมูลส่วนบุคคลที่จะถูกเปิดเผยแก่บุคคลภายนอกโดยไม่ได้รับอนุญาตจากผู้ใช้บริการ

ผู้ให้บริการจะรักษาข้อมูลส่วนบุคคลของผู้ใช้บริการไว้เป็นความลับ และจะใช้ข้อมูลส่วนบุคคลดังกล่าว เพื่อการให้บริการและพัฒนา Provider ID เท่านั้น ข้อมูลส่วนบุคคลของผู้ใช้บริการจะได้รับการคุ้มครองตามกฎหมายที่เกี่ยวข้อง และจะไม่เปิดเผยแก่บุคคลภายนอกโดยไม่ได้รับอนุญาตจากผู้ใช้บริการ

เว้นแต่ในกรณีดังต่อไปนี้เป็นไปตามกฎหมายหรือกฎระเบียบที่เกี่ยวข้อง

  1. เป็นการจำเป็นเพื่อปฏิบัติตามคำร้องขอของหน่วยงานราชการหรือหน่วยงานอื่นที่มีอำนาจตามกฎหมาย
  2. เป็นการจำเป็นเพื่อปกป้องสิทธิหรือความปลอดภัยของผู้ใช้บริการหรือบุคคลอื่น
  3. เป็นการจำเป็นเพื่อดำเนินการตามสัญญาระหว่างผู้ให้บริการและผู้ใช้บริการ

ข้อกำหนดและเงื่อนไขการให้บริการ Provider ID (Terms and Conditions for Provider ID)

คำนิยาม

ระบบเว็บไซต์พอร์ทัล Provider ID หมายถึง ระบบข้อมูลเจ้าหน้าที่ผู้ให้บริการ บน Digital Health Platform ให้บริการรหัสประจำตัวบุคลากรทางการแพทย์ มุ่งเน้นให้บุคลากรทางการแพทย์สามารถเข้าถึงข้อมูลผู้ป่วยบนแพลตฟอร์มกลางได้อย่างสะดวกและรวดเร็ว สอดคล้องกับนโยบาย “ยกระดับ 30 บาท รักษาทุกที่ ด้วยบัตรประชาชนใบเดียว” ผ่านระบบบูรณาการข้อมูลระหว่างหน่วยงานของรัฐ ภายใต้พระราชบัญญัติการบริหารงานและการให้บริการภาครัฐผ่านระบบดิจิทัล พ.ศ. 2562

ผู้ให้บริการ หมายถึง กระทรวงสาธารณสุข ซึ่งมีหน้าที่บำรุงรักษา พัฒนา และวางกรอบรายละเอียดของ Provider ID ภายใต้พระราชบัญญัติการบริหารงานและการให้บริการภาครัฐผ่านระบบดิจิทัล พ.ศ. 2562

ผู้ใช้บริการ หมายความถึง บุคคลธรรมดา นิติบุคคล หรือหน่วยงานใด ๆ ที่เข้าใช้บริการ Provider ID

บริการ หมายความรวมถึง การใช้ผลิตภัณฑ์ ซอฟต์แวร์และเว็บไซต์ของผู้ให้บริการ ที่เป็นส่วนหนึ่งของ Provider ID ซึ่งจัดทำขึ้นเพื่อเผยแพร่ข้อมูล ภายใต้พระราชบัญญัติการบริหารงานและการให้บริการภาครัฐผ่านระบบดิจิทัล พ.ศ. 2562

เจ้าหน้าที่ผู้ให้บริการ หมายความรวมถึง บุคลากรทางการแพทย์

1. ข้อกำหนด

1.1 ผู้ให้บริการให้บริการแก่ผู้ใช้บริการภายใต้ข้อกำหนดและนโยบายการให้บริการที่กำหนดไว้ในเอกสารฉบับนี้ ซึ่งต่อไปจะเรียกว่า “ข้อกำหนดทั่วไป” และข้อกำหนดเฉพาะที่ผู้ให้บริการกำหนด เพิ่มเติมสำหรับบริการระบบข้อมูลเจ้าหน้าที่ผู้ให้บริการ บน Digital Health Platform ผ่านทางเว็บไซต์ https://id.moph.go.th/

1.2 ผู้ใช้บริการมีหน้าที่จะต้องอ่านข้อกำหนดทั่วไปและข้อกำหนดเฉพาะให้เข้าใจโดยละเอียดก่อนเข้าใช้บริการ โดยข้อกำหนดทั่วไปและข้อกำหนดเฉพาะถือเป็นสาระสำคัญในการให้บริการของผู้ให้บริการ และมีผลผูกพันทางกฎหมายระหว่างผู้ใช้บริการและผู้ให้บริการ ในกรณีที่ข้อกำหนดทั่วไปมีข้อความขัดแย้งกับข้อกำหนดเฉพาะ ให้ข้อความในข้อกำหนดเฉพาะบังคับใช้สำหรับบริการนั้นแทนข้อกำหนดทั่วไป ผู้ให้บริการมีสิทธิในการแก้ไขเพิ่มเติมข้อกำหนดได้โดยไม่จำเป็นต้องแจ้งให้ผู้ใช้บริการทราบล่วงหน้าเป็นลายลักษณ์อักษร

1.3 ผู้ใช้บริการมีหน้าที่ปฏิบัติตามกฎหมาย กฎ ระเบียบ ข้อบังคับ หลักปฏิบัติที่เกี่ยวข้อง และข้อตกลงที่ผู้ให้บริการกำหนด

2. การยอมรับข้อกำหนด

2.1 ผู้ใช้บริการจะต้องอ่านและยอมรับข้อกำหนด ก่อนเข้าใช้บริการ หากไม่ยอมรับข้อกำหนดจะไม่สามารถใช้บริการได้

2.2 ผู้ใช้บริการถือว่าได้ยอมรับข้อกำหนดการใช้บริการของผู้ให้บริการโดยปริยาย เมื่อเริ่มใช้บริการ

2.3 ห้ามมิให้ผู้ใช้บริการนำรายชื่อบุคคลอื่นเข้าสู่ระบบ หรือ ปลอมแปลงรายชื่อเป็นบุคคลอื่นโดยมิได้รับความยินยอมจากบุคคลดังกล่าว ผู้ใช้บริการต้องจัดให้ผู้ให้ความยินยอมอ่านและยอมรับข้อกำหนดนี้ก่อน จึงจะนำรายชื่อเข้าสู่ Provider ID หากผู้ใช้บริการนำรายชื่อผู้ให้ความยินยอมเข้าสู่ Provider ID ผู้ให้บริการ ถือว่าผู้ใช้บริการได้รับรองแล้วว่าได้จัดให้ผู้ให้ความยินยอมอ่านและยอมรับข้อกำหนดนี้แล้ว และผู้ใช้บริการตกลงโดยปริยายว่าจะรับผิดในความเสียหายที่เกิดขึ้นสืบเนื่องจากการไม่อ่านหรือไม่ยอมรับข้อกำหนดนี้

2.4 ผู้ใช้บริการควรบันทึกหรือทำสำเนาข้อกำหนดทั่วไปไว้เพื่อเป็นหลักฐาน

3. การให้บริการโดยผู้ให้บริการ

3.1 ผู้ให้บริการอาจปรับปรุงหรือพัฒนารูปแบบและลักษณะของบริการได้ตลอดเวลา เพื่อประโยชน์สูงสุดของผู้ใช้บริการ โดยอาจแจ้งให้ผู้ใช้บริการทราบล่วงหน้าหรือไม่ได้แจ้งก็ได้

3.2 ผู้ให้บริการมีสิทธิหยุดให้บริการชั่วคราวหรือถาวรหรือยกเลิกการให้บริการแก่ผู้ใช้บริการบางรายได้โดยไม่ต้องแจ้งผู้ใช้บริการทราบล่วงหน้า ในกรณีที่การให้บริการดังกล่าวส่งผลกระทบต่อผู้ใช้บริการอื่น ๆ หรือขัดต่อกฎหมาย โดยผู้ให้บริการจะแจ้งให้ผู้ใช้บริการทราบภายใน 15 วันหลังจากดำเนินการ โดยไม่ชดเชยค่าเสียหายใด ๆ ทั้งสิ้น

3.3 เมื่อการให้บริการถูกระงับหรือยกเลิก ผู้ใช้บริการจะสูญเสียสิทธิในการเข้าถึงบริการ และข้อมูลทั้งหมดที่อยู่ในบัญชีของผู้ใช้บริการ

4. การใช้บริการโดยผู้ใช้บริการ

4.1 ผู้ใช้บริการต้องให้ข้อมูลส่วนบุคคลที่ถูกต้องและเป็นปัจจุบันแก่ผู้ให้บริการ อันเป็นส่วนหนึ่งของกระบวนการลงทะเบียนใช้บริการหรือการใช้บริการที่ต่อเนื่องหากมีการร้องขอ

4.2 ผู้ใช้บริการต้องใช้บริการนี้เพื่อวัตถุประสงค์ที่ได้รับอนุญาตตามข้อตกลงที่ได้ทำกับผู้ให้บริการ และไม่ขัดต่อกฎหมาย กฎ ระเบียบ ข้อบังคับ หลักปฏิบัติ ที่เป็นที่ยอมรับโดยทั่วไป ซึ่งรวมถึงการนำเข้าข้อมูลส่วนบุคคลของผู้ใช้บริการรายอื่นโดยไม่ได้รับความยินยอมจากเจ้าของข้อมูลส่วนบุคคล

4.3 ผู้ใช้บริการจะไม่เข้าใช้ หรือพยายามเข้าใช้บริการหนึ่งบริการใดโดยวิธีอื่นนอกจากช่องทางที่ผู้ให้บริการจัดเตรียมไว้ให้ เว้นแต่ผู้ใช้บริการจะได้รับอนุญาตจากผู้ให้บริการโดยชัดแจ้งให้ทำเช่นนั้นได้

4.4 ผู้ใช้บริการต้องเข้าใช้หรือพยายามเข้าใช้บริการหนึ่งบริการใดผ่านช่องทางที่ผู้ให้บริการจัดเตรียมไว้ให้เท่านั้น เว้นแต่ผู้ใช้บริการจะได้รับอนุญาตจากผู้ให้บริการโดยชัดแจ้งให้ทำเช่นนั้นได้

4.5 ผู้ใช้บริการต้องไม่ทำสำเนา คัดลอก ทำซ้ำ ขาย แลกเปลี่ยน หรือขายต่อบริการโดยไม่ได้รับอนุญาตจากผู้ให้บริการ ซึ่งอาจเป็นการละเมิดสิทธิของผู้ใช้บริการอื่น

4.6 ผู้ใช้บริการต้องเป็นผู้รับผิดชอบแต่เพียงผู้เดียวในความเสียหายที่เกิดขึ้นต่อบุคคลใด ๆ อันเนื่องมาจากการละเมิดข้อกำหนด

4.7 ผู้ให้บริการอาจกำหนดขีดจำกัดสูงสุดในปริมาณข้อมูลและพื้นที่จัดเก็บที่ผู้ใช้สามารถใช้งานบริการได้ โดยขีดจำกัดสูงสุดดังกล่าวจะขึ้นอยู่กับการพิจารณาของผู้ให้บริการ เว้นแต่ผู้ให้บริการจะตกลงเป็นลายลักษณ์อักษรเป็นอย่างอื่น

5. รหัสผ่านและการรักษาความปลอดภัยของบัญชี

กรณีที่มีการสร้างบัญชีผู้ใช้งานสำหรับใช้งานในวัตถุประสงค์ต่างๆ ผู้ใช้งานมีหน้าที่ในการรักษาความลับของรหัสผ่านที่เชื่อมโยงกับบัญชีใด ๆ ที่ใช้ในการเข้าถึงบริการและจะเป็นผู้รับผิดชอบแต่เพียงผู้เดียวต่อผู้ให้บริการ สำหรับการใช้งานทั้งหมดที่เกิดขึ้นภายใต้บัญชีของผู้ใช้บริการ

6. การรักษาความเป็นส่วนตัวและข้อมูลส่วนบุคคลของผู้ใช้บริการ

6.1 ผู้ให้บริการจะปฏิบัติต่อข้อมูลส่วนบุคคลและปกป้องความเป็นส่วนตัวของผู้ใช้บริการและเก็บรักษาข้อมูลส่วนบุคคลดังกล่าวตามนโยบายส่วนบุคคลที่ผู้ให้บริการกำหนดไว้

6.2 ผู้ให้บริการจะกำกับดูแลให้ผู้ประมวลผลข้อมูลดำเนินการประมวลผลข้อมูลส่วนบุคคลที่สอดคล้องตามนโยบายส่วนบุคคลของผู้ให้บริการ

6.3 ผู้ใช้บริการมีหน้าที่ปฏิบัติตามพระราชบัญญัติคุ้มครองข้อมูลส่วนบุคคล พ.ศ. 2562 และที่แก้ไขเพิ่มเติม รวมทั้งกฎ ระเบียบ ของกระทรวงสาธารณสุข และระเบียบของทางราชการที่เกี่ยวข้อง

7. เนื้อหาในบริการ

7.1 ผู้ใช้บริการอาจได้รับ เข้าถึง สร้าง ส่ง หรือแสดงข้อมูล เช่น ไฟล์ข้อมูล ข้อความลายลักษณ์อักษร ซอฟต์แวร์คอมพิวเตอร์ โดยเป็นส่วนหนึ่งของบริการหรือโดยผ่านการใช้บริการ ซึ่งต่อไปนี้จะเรียกว่า “เนื้อหา”

7.2 เนื้อหาที่นำเสนอต่อผู้ใช้บริการ อาจได้รับการคุ้มครองโดยสิทธิในทรัพย์สินทางปัญญาของเจ้าของเนื้อหานั้น หรือโดยกฎหมายที่ใช้บังคับกับผู้ให้บริการ ผู้ใช้บริการไม่มีสิทธิเปลี่ยนแปลงแก้ไข จำหน่ายจ่ายโอน หรือสร้างผลงานต่อเนื่องโดยอาศัยเนื้อหาดังกล่าว เว้นแต่ได้รับอนุญาตจากเจ้าของทรัพย์สินทางปัญญาหรือผู้ให้บริการ ตามที่กฎหมายกำหนด

7.3 ผู้ให้บริการมีสิทธิในการคัดกรอง ตรวจทาน ทำเครื่องหมาย เปลี่ยนแปลงแก้ไข ปฏิเสธ หรือลบเนื้อหาใด ๆ ที่ไม่เหมาะสมออกจากบริการ ซึ่งรวมถึงการจัดเตรียมเครื่องมือในการคัดกรองเนื้อหาอย่างชัดเจน เป็นไปตามกฎหมาย กฎ ระเบียบของทางราชการที่เกี่ยวข้อง

7.4 ผู้ให้บริการไม่รับผิดชอบต่อความเสียหายใด ๆ ไม่ว่าทางตรงหรือทางอ้อม อันเกิดจากเนื้อหาที่สร้างขึ้นโดยบุคคลอื่น ผู้สร้างเนื้อหาดังกล่าวจะต้องเป็นผู้รับผิดชอบต่อความเสียหายที่เกิดขึ้นทั้งหมด

7.5 ผู้ใช้บริการใดก็ตามที่ได้ทำการสร้าง เผยแพร่ หรือแสดงเนื้อหาใด ๆ ในบริการ จะต้องเป็นผู้รับผิดชอบต่อความเสียหายใด ๆ ที่เกิดขึ้นจากเนื้อหาที่ตนสร้าง เผยแพร่ หรือแสดงในบริการทั้งหมด
